🌐 라우터 포트포워딩 설정 가이드

1️⃣ 라우터 관리자 페이지 접속
   - 브라우저에서 192.168.1.1 또는 192.168.0.1 접속
   - 또는 라우터 하단의 IP 주소 확인
   - 관리자 계정/비밀번호 입력

2️⃣ 포트포워딩 설정 찾기
   - "포트포워딩" 또는 "Port Forwarding"
   - "포트매핑" 또는 "Port Mapping"
   - "NAT 설정" 또는 "NAT Configuration"

3️⃣ 새 규칙 추가
   - 외부 포트: 5000
   - 내부 포트: 5000
   - 프로토콜: TCP
   - 대상 IP: 192.168.228.129
   - 설명: Flask Bank App

4️⃣ 설정 저장 및 적용

5️⃣ 외부 IP 확인
   - 인터넷에서 "내 IP 확인" 검색
   - 또는 whatismyip.com 접속

6️⃣ 접속 테스트
   - 외부에서: http://[외부IP]:5000
   - 예: http://123.456.789.123:5000

⚠️ 보안 주의사항:
   - 강력한 비밀번호 사용
   - 사용 후 서버 종료
   - 방화벽 설정 확인
